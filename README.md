# 期末刷题宝典

英语期末复习刷题网站，从 Word 题库自动解析生成，支持浏览器直接打开使用。

## 初衷

期末考题是 Word，答案背完了但心里没底——背了和会了是两码事。怎么检验？总不能对着文档自己考自己吧。

突然想起科目一，驾考宝典那种选完即出结果、刷到条件反射的快感记忆犹新。于是就把期末题库变成了驾考宝典，选完即判，错题收录，刷到每道题刻进 DNA。

## 使用方式

双击打开 `index.html` 即可，无需安装任何依赖。

## 功能

- **4种题型**：听力(40)、段落匹配(60)、选词填空(8套)、阅读(82)，共190题
- **4种模式**：顺序练习、随机刷题、错题本、收藏夹
- **即选即判**：选择答案后立即显示对错及正确答案翻译
- **题目导航**：点击底部题号弹出导航面板，可跳转到任意题目
- **数据持久化**：答题记录、错题、收藏保存在浏览器 localStorage，刷新不丢失

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 刷题网站主页面 |
| `data.js` | 题库数据（由解析脚本生成） |
| `parse.py` | 题库解析脚本，从 docx 提取题目生成 JSON |
| `questions.json` | 解析后的完整题库 JSON |
| `assets/` | 原始 Word 题库文件 |

## 重新生成题库

当 Word 题库有更新时，运行解析脚本重新生成 `data.js`：

```bash
pip install python-docx
python parse.py
```

脚本会读取 `assets/` 下的四个 docx 文件，输出 `questions.json` 和 `data.js`。

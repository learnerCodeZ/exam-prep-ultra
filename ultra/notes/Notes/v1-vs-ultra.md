# v1.0 vs Ultra 版本对比

## 架构差异

| | v1.0 Classic | Ultra |
|--|-------------|-------|
| **代码组织** | 单文件（index.html 649行，含全部 JS/CSS） | 分离（HTML 171行 + app.js 980行 + parser-md.js 459行 + CSS） |
| **题库数据** | data.js 内嵌（620KB，全部写死在 JS 里） | default.json 外置 + 用户导入存 localStorage |
| **题库管理** | 无（只有一套固定题库） | 多题库：切换、重命名、删除、追加 |
| **解析器** | 无（数据手动用 Python 脚本生成） | 浏览器端实时解析 Word/PDF/MD/粘贴文本 |

## 功能差异

| 功能 | v1.0 | Ultra |
|------|------|-------|
| 题型 | 4种固定（听力/匹配/填空/阅读） | 动态生成，支持单选/多选/判断 |
| 导入题库 | ❌ | ✅ 文件上传 + 粘贴文本 |
| 追加题目 | ❌ | ✅ 导入到现有题库 + 侧栏+按钮 |
| 多题库切换 | ❌ | ✅ 侧栏切换 |
| 重命名题库 | ❌ | ✅ |
| 删除题库 | ❌ | ✅ |
| 答案编辑 | ❌ | ✅ 修改答案 + 恢复默认 |
| 无答案题 | 不支持 | ✅ 显示"无答案"不计对错 |
| 帮助弹窗 | ❌ | ✅ 右上角?按钮 |
| 键盘快捷键 | 基础 | ✅ 数字键/字母键/方向键/F收藏 |
| 错题本 | ✅ | ✅ |
| 收藏夹 | ✅ | ✅ |
| 题目导航 | ✅ | ✅ |

## 代码量对比

- v1.0：**22 个函数**（纯刷题渲染）
- Ultra：**55 个函数**（刷题 + 导入 + 题库管理 + 答案编辑 + 帮助）

## 总结

v1.0 是一个**只能刷固定题库的纯刷题工具**，数据写死在代码里。Ultra 在此基础上加了完整的题库生命周期（导入→管理→刷题→编辑），从"看题工具"变成了"题库平台"。
